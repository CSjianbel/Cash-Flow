# Stock-Trading-CLI-Simulator

A bash program that simulates stock trading. 

## Features

1. View balance (initially $10,000)
2. View transactions
3. View portfolio
4. Fetch stock price
5. Buy stocks 
6. Sell stocks 

## Clone

```bash
git clone 
```

## Usage 

```bash
cd src 

chmod +x ./stock_trader.sh 

./stock_trader.sh
```
